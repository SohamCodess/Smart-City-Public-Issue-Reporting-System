# 🌆 Smart City Public Issue Reporting System

## 📌 Description
Smart City Public Issue Reporting System is a web-based application that allows citizens to report public issues such as road damage, water leakage, garbage problems, etc.  
The system helps authorities track and resolve issues efficiently.

---

## 🚀 Features
- User can report public issues
- Add issue details and description
- Simple and user-friendly interface
- Helps track city problems
- Improves communication between citizens and authorities

---

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python  
- **Database:** MongoDB  

---

## ⚙️ How It Works
1. User opens the website  
2. Submits issue details  
3. Data is stored in database  
4. Authorities can view and take action
